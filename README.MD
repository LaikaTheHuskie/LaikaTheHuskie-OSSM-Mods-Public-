## 🛠️ The G.I.R.T.H. Mounting System (V1)  
Generalized Interchangeable Retaining & Toy Handling  

Standard plates are too small; this system is built for **maximum girth** with minimum profile.  
Developed for the OSSM, this is a heavy-duty, universal mounting solution designed to support toys up to 6" in diameter with bases up to 10". (and is scalable even further)


------------------------------
## 📏 How to Choose Your Parts
Follow these three measurements to identify which STL files you need to print.
## Step 1: The Backplate (Base Width (Diamatre))
Measure the widest part of the toy's base.

* 0 – 74mm (3") → Print BACK_PLATE_01
* 0 – 100mm (4") → Print BACK_PLATE_02
* 0 – 126mm (5") → Print BACK_PLATE_03
* 0 – 151mm (6") → Print BACK_PLATE_04
* Note: Plates 05 (7") and 06 (8") are available for extreme builds., they do not have retaining rings becasue i have suspicions that they will never be used
* Note: At the time of writing the Readme, V1 backplates have been reired and current version is V2 with multihole arrangement, each step supports the previous plates PCD
* Note: V3 plates will have less holes overall and more structure, but i strongly believe V2 plates will be strong enough for most applications


## Step 2: The Retaining Ring (Shaft Width (Diamatre))
Measure the width of the toy's shaft where the ring will sit. Match the Ring number to your Backplate.

| If using Backplate... | Choose Ring Size (Shaft Clearance) |
|---|---|
| 01 | 1" / 1.5" / 2" / 2.5" |
| 02 | 2" / 2.5" / 3" / 3.5" |
| 03 | 3" / 3.5" / 4" / 4.5" |
| 04 | 4" / 4.5" / 5" / 5.5" |

## Step 3: The Spacers (Base Height)
Measure the thickness/height of the toy's base. Use this formula to ensure a secure grip:

(Base Height - 0.5") = Required Spacer Stack


* Available Spacers: 1/4", 5/16", 3/8", 1/2", 5/8", 3/4", 1"
* Pro Tip: Place spacers behind the plate to shorten the bolt reach if your M10s are too long. ensure plate is rotated to clear machine housing

------------------------------
## ⚙️ Recommended Print Settings
To handle high torque and reciprocating mass, it is not advisable to use standard prining profiles:
(Suggested possible print settings)
* Material: PETG or Tough PLA (ABS/ASA preferred for high-stress).
* Walls: 4–6 Perimeters.
* Infill: 40%+ (Gyroid or Cubic).
* Top/Bottom: 5 layers minimum.
* These plates and the rings have been tested using basic PLA on a bambu P1S using .20 strength preset with no additional changes, individual milage may vary 



------------------------------
## 🔩 Hardware Specs

* Bolts: Metric M10 Bolts.
* Length: 50mm to 200mm recommended.
* Files: .STEP (raw geometry), .DXF (hole patterns), and .STL (print-ready).
