## 🛠️ The G.I.R.T.H. Mounting System (V1)
Generalized Interchangeable Retaining & Toy Handling
Standard plates are too small; this system is built for maximum girth. Developed for the OSSM community, this is a heavy-duty, universal mounting solution designed to support toys up to 6" in diameter with bases up to 10".
------------------------------
## 📏 How to Choose Your Parts
Follow these three measurements to identify which STL files you need to print.
## Step 1: The Backplate (Base Width)
Measure the widest part of the toy's base.

* 0 – 74mm (3") → Print BACK_PLATE_01
* 74 – 100mm (4") → Print BACK_PLATE_02
* 100 – 126mm (5") → Print BACK_PLATE_03
* 126 – 151mm (6") → Print BACK_PLATE_04
* Note: Plates 05 (7") and 06 (8") are available for extreme builds.

## Step 2: The Retaining Ring (Shaft Width)
Measure the width of the toy's shaft where the ring will sit. Match the Ring number to your Backplate.

| If using Backplate... | Choose Ring Size (Shaft Clearance) |
|---|---|
| 01 | 1" / 1.5" / 2" / 2.5" |
| 02 | 2" / 2.5" / 3" / 3.5" |
| 03 | 3" / 3.5" / 4" / 4.5" |
| 04 | 4" / 4.5" / 5" / 5.5" |

## Step 3: The Spacers (Base Height)
Measure the thickness/height of the toy's base. Use this formula to ensure a secure grip:

(Base Height - 0.5") = Required Spacer Stack


* Available Spacers: 1/4", 5/16", 3/8", 1/2", 5/8", 3/4", 1"
* Pro Tip: Place spacers behind the plate to shorten the bolt reach if your M10s are too long.

------------------------------
## ⚙️ Recommended Print Settings
To handle high torque and reciprocating mass, do not use standard profiles:

* Material: PETG or Tough PLA (ABS/ASA preferred for high-stress).
* Walls: 4–6 Perimeters.
* Infill: 40%+ (Gyroid or Cubic).
* Top/Bottom: 5 layers minimum.

------------------------------
## 🔩 Hardware Specs

* Bolts: Metric M10 Bolts.
* Length: 50mm to 200mm recommended.
* Storage: Includes Gridfinity bolt and hardware holders for organization.
* Files: .STEP (raw geometry), .DXF (hole patterns), and .STL (print-ready).
